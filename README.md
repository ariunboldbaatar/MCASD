# Next.js SCADA Dashboard Template

Basic Next.js + Tailwind template for your SCADA dashboard. Deploy this to Vercel by pushing to GitHub and importing the repo in Vercel.

**How to use:**
1. Unzip and upload files to a GitHub repository.
2. Push to GitHub and import the repo into Vercel (select Next.js framework preset).
3. Vercel will install dependencies and build the site.
